# Contributing to llms-txt-templates

Contributions are welcome. This project improves when practitioners share templates from real-world implementations.

## How to Contribute

**Submit a new template:**
Create a new `.txt` file in `templates/` following the format of existing templates. Submit a pull request with a brief description of the use case.

**Improve the spec or validator:**
Open an issue describing the proposed change and why it matters.

**Report a bug in the validator:**
Open an issue with the input that caused the problem and the expected vs actual output.

## Guidelines

- Follow the llms.txt spec format. Templates must be valid according to the specification.
- Keep templates realistic. Use plausible (but fictional) company data.
- Avoid marketing language. This is a technical resource.
- Write in English for global accessibility.

## Code of Conduct

Be professional and constructive. We're here to build a useful resource for the GEO community.
